Question-1: What are the different ways to select an element in the DOM?

We are able to select elements by multiple ways. The common ways are document.getElementById(), document.getElementsByClassName(), document.getElementsByTagName(), document.querySelector(), document.querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML returns the HTML content inside an element. innerText returns the visible text content of an element. textContent returns all the text content inside an element including hidden text.

Question-3: What is event delegation in the DOM?

Event delegation means adding one event listener to a parent element instead of adding it to each child element. It works because events bubble up from the child to the parent, so the parent can handle the event for all its children.

Question-4: What is event bubbling in the DOM?

when you click on something, the event first happens on that element, and then it "bubbles up" to its parent elements, one by one, until it reaches the top.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element, we use document.createElement('tag'). To add or remove it, we use appendChild() to add and removeChild() to remove from the parent element.
